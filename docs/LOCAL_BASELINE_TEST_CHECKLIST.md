# چک‌لیست تست لوکال فاز ۰ — DeDe

## هدف سند
این سند برای ثبت رفتار فعلی لوکال قالب و افزونه‌های DeDe قبل از شروع فازهای refactor و bug-fix تهیه شده است. هدف این است که وضعیت فعلی سایت، مسیرهای اصلی، قابلیت‌های خرید، پنل مشتری، افزونه‌های اختصاصی و بخش‌های مدیریتی به صورت دستی ثبت شود تا بعداً بتوان رفتار هر مرحله را با آن مقایسه کرد.
